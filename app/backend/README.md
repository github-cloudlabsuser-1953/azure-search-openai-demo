# `app/backend` ディレクトリ

`app/backend` ディレクトリは、このプロジェクトのバックエンド部分を含んでいます。主に Python で書かれており、Azure Search、Azure Blob Storage、OpenAI などのサービスとのインタラクションを管理します。

## 主なファイルとディレクトリ

- `app.py`: アプリケーションのエントリーポイントです。ここでは、Azure サービスとの接続や OpenAI とのインタラクションが設定されています。また、各種エンドポイントもここで定義されています。

- `approaches/`: このディレクトリには、異なるアプローチを実装する Python ファイルが含まれています。これらのアプローチは、ユーザーのクエリを処理し、適切な応答を生成するために使用されます。

- `config.py`: このファイルには、アプリケーションの設定が含まれています。これには、Azure サービスや OpenAI との接続情報などが含まれています。

## ローカルでの実行

`start.sh` スクリプトを使用して、ローカルでバックエンドを実行できます。このスクリプトは、必要な Python パッケージをインストールし、アプリケーションを起動します。

```sh
./start.sh
```

アプリケーションが起動したら、`http://localhost:5000` にアクセスして、アプリケーションを使用できます。

## テスト

バックエンドのテストを実行するには、`pytest` を使用します。以下のコマンドを実行すると、すべてのテストが実行されます。

```sh
pytest
```

テストには、`pytest` が必要です。インストールされていない場合は、以下のコマンドでインストールできます。

```sh
pip install pytest
```

# `app/frontend` ディレクトリ

`app/frontend` ディレクトリには、このプロジェクトのフロントエンド部分が含まれています。フロントエンドは、ユーザーとのインタラクションを管理し、バックエンドとの通信を処理します。

## 主なファイルとディレクトリ

- `src/`: このディレクトリには、フロントエンドのソースコードが含まれています。主に TypeScript で書かれており、React を使用して構築されています。

- `public/`: このディレクトリには、フロントエンドの HTML ファイルや画像などの静的ファイルが含まれています。
